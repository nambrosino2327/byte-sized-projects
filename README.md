# Byte-Sized Projects
*By Nathan Ambrosino*

---

This repo serves as a collection of smaller projects that I did for fun. Each of them are too small for their own individual repo.

# List of Projects
- **The C-Qualizer:** An equalizer effect made with one C source file. You can customize the height of each equalizer bar, how many bars are drawn, and how long they last before the next frame is drawn.
